Data Analysis of Game FIFA 15 - FIFA 22 Player Dataset with Python

The FIFA video game series is one of the most popular sports video game franchises in the world. The game has been released annually since 1993, and each new release features updated player ratings and statistics. This dataset contains player data from FIFA 15 to FIFA 22, including player name, age, nationality, position, overall rating, and other attributes. Dataset that I used is taken from Kaggle.

Technologies: Python

Libraries: pandas, matplotlib
